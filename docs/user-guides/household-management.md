## Getting Started

### Creating a Household

1. **Launch PetForce** and complete registration
2. **Navigate to Onboarding** (automatic after signup)
3. **Click "Create a Household"**
4. **Enter Details:**
   - **Name:** 2-50 characters (e.g., "The Smith Family")
   - **Description (Optional):** Brief description (e.g., "2 dogs, 1 cat")
5. **Click "Create"**

**You're now the Household Leader!** You'll receive a unique invite code.

## Inviting Members

### Method 1: Share Invite Code (Manual Entry)

1. **Go to Household Settings** ‚Üí **Invite Code**
2. **Copy your code** (e.g., `SMITH-ALPHA-BRAVO`)
3. **Share via:**
   - Text message
   - Chat app
   - Write it down
4. **Recipient enters code** in PetForce app ‚Üí "Join a Household"

### Method 2: QR Code (In-Person)

1. **Go to Household Settings** ‚Üí **QR Code**
2. **Show QR code** on your screen
3. **Recipient scans** with PetForce app camera
4. **Code auto-fills** - they just tap "Submit"!

**Perfect for:** In-person meetings, printed materials

### Method 3: Email Invite (Remote)

1. **Go to Household Settings** ‚Üí **Invite Members**
2. **Enter recipient email** and optional message
3. **Click "Send Invite"**
4. **Recipient receives** branded email with:
   - Direct join link
   - Manual invite code option
   - Household details

**Perfect for:** Remote family members, pet sitters

## Joining a Household

### Via Invite Code

1. **Open PetForce** and complete registration
2. **Click "Join a Household"** on onboarding screen
3. **Enter invite code** (e.g., `SMITH-ALPHA-BRAVO`)
4. **Click "Submit Request"**
5. **Wait for approval** from household leader

**Note:** You'll receive a push notification when approved!

### Via QR Code (Mobile)

1. **Open PetForce** app on mobile
2. **Click "Join a Household"** ‚Üí **"Scan QR Code"**
3. **Point camera** at QR code
4. **Code auto-fills** ‚Üí click "Submit Request"
5. **Wait for approval**

### Via Email Link

1. **Open email invite** from household leader
2. **Click "Join [Household Name]"** button
3. **Login/signup** if needed
4. **Confirm join request**
5. **Wait for approval**

## Managing Join Requests (Leaders Only)

### Approving Requests

1. **Go to Household** ‚Üí **Pending Requests** tab
2. **Review request** details:
   - Requester name/email
   - Request time
3. **Click "Approve"**
4. **Member added instantly!** They get a push notification

### Rejecting Requests

1. **Go to Pending Requests**
2. **Click "Reject"** on request
3. **Requester notified** (can try again with new code)

**Tip:** Communicate via other channels if rejecting to avoid confusion!

---

## Member Management (Leaders Only)

### Removing a Member

1. **Go to Household** ‚Üí **Members** tab
2. **Find member** to remove
3. **Click "Remove Member"**
4. **Confirm action**
5. **Member removed** and receives notification

**Important:** Removed members lose access immediately. Their session is invalidated.

### Temporary Access (Pet Sitters, Guests)

**Create Temporary Member:**
1. **Mark "Temporary" when approving** join request
2. **Set expiration date** (e.g., 7 days for vacation pet sitter)
3. **Member auto-removed** after expiration

**Extend Temporary Access:**
1. **Go to Members** ‚Üí **Temporary Member**
2. **Click "Extend Access"**
3. **Set new expiration date**
4. **Save changes**

**Use Cases:**
- Vacation pet sitters (7-14 days)
- Trial period for new members (30 days)
- Temporary caregivers (varies)

## Household Security

### Regenerating Invite Code

**When to regenerate:**
- Code shared publicly by mistake
- Former member has old code
- Periodic security refresh

**How to regenerate:**
1. **Go to Household Settings** ‚Üí **Invite Code**
2. **Click "Regenerate Code"**
3. **Old code invalidated** immediately
4. **Share new code** with trusted members only

**Rate Limit:** 5 regenerations per hour (prevents abuse)

### Leadership Transfer

**Leaving as Leader:**
1. **Go to Household Settings** ‚Üí **Leave Household**
2. **Choose successor** (optional):
   - Select a member to become new leader
   - Or auto-promote longest-standing member
3. **Click "Leave"**
4. **Leadership transferred** immediately

**Important:** Only the leader can leave and transfer leadership. Regular members can leave anytime without restrictions.

## Troubleshooting

### "Invalid Invite Code" Error

**Causes:**
- Typo in code entry
- Code expired (30 days old)
- Code was regenerated by leader

**Solution:**
1. **Double-check spelling** (codes are UPPERCASE)
2. **Ask leader** to share code again or regenerate
3. **Try QR code** or email invite method instead

### "Already in a Household" Error

**Cause:** You're already a member of another household

**Solution:**
1. **Leave current household** first
2. **Then join new household**

**Note:** Phase 1 limits users to 1 household. Multi-household support coming in Phase 2!

### "Household at Capacity" Error

**Cause:** Household has reached 15-member limit

**Solution:**
1. **Ask leader** to remove inactive members
2. **Or create a new household** and invite active members

## Best Practices

### For Leaders

1. **Choose clear household names** (e.g., "Smith Family Pets" not "House123")
2. **Add description** with pet count for context
3. **Review requests promptly** (< 24 hours ideal)
4. **Use temporary access** for pet sitters (don't leave them as permanent)
5. **Regenerate codes quarterly** for security

### For All Members

1. **Enable push notifications** for instant updates
2. **Don't share invite codes** publicly or on social media
3. **Update your profile** with clear name/photo (helps leader identify you)
4. **Communicate** with household before joining (avoid surprise requests)

## FAQ

**Q: How many households can I join?**
A: Currently 1 household per user (Phase 1). Multi-household support coming soon!

**Q: Can I have multiple leaders?**
A: No, households have 1 leader. The leader can transfer leadership to another member when leaving.

**Q: How long do invite codes last?**
A: 30 days by default. Leaders can regenerate codes anytime.

**Q: Can I withdraw a join request?**
A: Yes! Go to Pending Requests and click "Withdraw" before the leader responds.

**Q: What happens to my data if I leave a household?**
A: Your contributions remain visible to the household, but you lose access to view/edit household data.

**Q: Can removed members rejoin?**
A: Yes, if they have a valid invite code and the leader approves their new request.
