**Feature**: Cohort Analysis Capability  
**Owner**: Ana (Analytics)  
**Designer**: Ana (with Dexter review)  
**Version**: 1.0  
**Date**: 2026-01-25

## Overview

This document provides detailed wireframes for the Cohort Analysis dashboard. All designs follow PetForce's family-first, simplicity-focused product vision.

## Design Principles

1. **Scannable at a Glance**: KPIs first, charts second, details on demand
2. **Mobile-First**: Design for small screens, scale up for desktop
3. **Accessible**: WCAG AA compliant, colorblind-safe, keyboard navigable
4. **Progressive Disclosure**: Show summary, reveal details on interaction
5. **Actionable Insights**: Every chart answers "What should I do?"

## Accessibility Annotations

### Keyboard Navigation

**Heatmap**:
- `Tab`: Focus on heatmap
- `Arrow Keys`: Navigate cells
- `Enter`: View cohort details
- `Esc`: Close tooltip

**Line Chart**:
- `Tab`: Focus on chart
- `Arrow Keys`: Navigate data points
- `Space`: Toggle legend item
- `Enter`: View details
- `Esc`: Close tooltip

### Hover Effects
- Transition duration: 150ms
- Easing: ease-in-out
- Properties: background-color, transform, opacity

### Loading Animations
- Skeleton screens (no spinners)
- Progressive rendering (show cached, update)
- Subtle pulse animation (opacity 0.5 â†” 1.0)
