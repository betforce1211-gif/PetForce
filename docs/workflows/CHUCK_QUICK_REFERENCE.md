# Chuck - Quick Reference Card

*Quality gates protect pet families. Every deployment matters.*

---

## One-Command Push

```bash
npm run chuck:push
```

**What it does:**
1. Validates branch ✓
2. Generates commit ✓
3. Runs quality gates ✓
4. Pushes to remote ✓
5. Creates PR ✓
6. Enables auto-merge ✓

---

## Essential Commands

```bash
# Start new work
./scripts/chuck create-branch feature PET-123 "description"

# Push everything
npm run chuck:push

# Manual validation
npm run chuck:validate    # Check branch name
npm run chuck:check       # Run quality gates
npm run chuck:pr          # Validate PR ready
```

## Branch Naming

```
Pattern: type/TICKET-ID-description

Types:
  feature/    → New features (→ develop)
  fix/        → Bug fixes (→ develop)
  hotfix/     → Critical fixes (→ main)
  docs/       → Documentation (→ develop)
  refactor/   → Refactoring (→ develop)
  test/       → Tests (→ develop)
  chore/      → Maintenance (→ develop)
  release/    → Release prep (→ main)

Examples:
  feature/PET-123-add-notifications
  fix/PET-456-login-bug
  hotfix/PET-789-data-loss
```

## Quality Gates

| Gate | Required | Command | Blocks |
|------|----------|---------|--------|
| Lint | ✓ | `npm run lint` | Yes |
| TypeCheck | ✓ | `npm run typecheck` | Yes |
| Build | ✓ | `npm run build` | Yes |
| Tests | - | `npm test` | No* |
| Security | - | `npm run security:audit` | No* |

*Will become required in future versions

## Workflow Phases

```
1. Validation → 2. Quality Gates → 3. Git Ops → 4. PR/Merge
   ↓                ↓                  ↓            ↓
 Branch           Lint              Fetch        Create PR
 Commits          TypeCheck         Rebase       Assign
 Changes          Test              Commit       Auto-merge
 Message          Security          Push
                  Build
```

## Quick Fixes

### Invalid Branch Name
```bash
git branch -m feature/PET-XXX-description
```

### Lint Errors
```bash
npm run lint -- --fix
npm run format
```

### Merge Conflicts
```bash
# Fix conflicts in editor
git add <files>
git rebase --continue
npm run chuck:push  # Retry
```

### GitHub CLI Not Setup
```bash
brew install gh
gh auth login
```
